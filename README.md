# Marketing Page

Eine einfache "Hallo Welt" Webseite erstellt mit Next.js und Tailwind CSS.

## Entwicklung

Installiere die Abhängigkeiten:

```bash
npm install
```

Starte den Entwicklungsserver:

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment auf Vercel

1. Push deinen Code zu GitHub
2. Gehe zu [vercel.com](https://vercel.com)
3. Importiere dein GitHub Repository
4. Vercel erkennt automatisch Next.js und deployt die Seite

Die Seite wird automatisch bei jedem Push zum Main-Branch neu deployed.
